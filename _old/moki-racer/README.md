# Móki Racer

Ein Rennspiel mit einem Eichhörnchen (Móki - ungarisch für Eichhörnchen), das auf einer Baumrinden-Schlitten den Hang hinunter fährt und Nüsse einsammelt.

## Installation

Da npm mit UNC-Pfaden unter WSL Probleme hat, führe die Installation direkt in deinem WSL-Terminal aus:

```bash
npm install
```

## Entwicklung

Starte den Development Server:

```bash
npm run dev
```

Das Spiel öffnet sich im Browser unter `http://localhost:5173`

## Steuerung

- **Pfeiltaste Links/Rechts**: Eichhörnchen steuern
- **Leertaste**: Springen

## Spielziel

Sammle so viele Nüsse wie möglich ein, während du den verschneiten Hang hinunter rast!

## Technologie

- **Three.js**: 3D-Grafik im Browser
- **Vite**: Schnelles Build-Tool
- **Vanilla JavaScript**: Keine zusätzlichen Frameworks

## Features

- ✅ Cartoon-Stil Grafik
- ✅ Eichhörnchen auf Baumrinden-Schlitten
- ✅ Nüsse zum Einsammeln
- ✅ Verschneiter Hang mit Bäumen und Felsen
- ✅ Physik-basierte Bewegung
- ✅ Kamera folgt dem Spieler
